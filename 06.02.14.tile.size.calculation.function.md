#### Tile Size Calculation Function

<div class="syntax">
calc_min_log2_tile_cols( ) {
    minLog2 = 0
    while ( (MAX_TILE_WIDTH_B64 << minLog2) < Sb64Cols )
        minLog2++
    return minLog2
}

</div>

<div class="syntax">
calc_max_log2_tile_cols( ) {
    maxLog2 = 1
    while ( (Sb64Cols >> maxLog2) >= MIN_TILE_WIDTH_B64 )
        maxLog2++
    return maxLog2 - 1
}

</div>
